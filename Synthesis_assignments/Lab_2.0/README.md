# Lab 02 – Log File Analysis and Error Debugging

## Objective
This lab focuses on generating a synthesis log file and learning how to search and debug errors and warnings.

---

## Tasks Covered

- Generate synthesis log file
- Search inside the log for errors and warnings
- Identify common issues such as:
  - Latch inference
  - Combinational loops
  - TCL script errors (e.g., wrong file names or missing characters)

---

## Key Learning Points

- How to read and filter log files efficiently
- Difference between errors and warnings
- Understanding common RTL and scripting mistakes
- Improving debugging skills in synthesis flow

---

## Conclusion
This lab helps in identifying and analyzing synthesis issues through log files, which is essential for debugging RTL designs and TCL scripts in ASIC flow.
